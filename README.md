# MailDownloaderNext

MailDownloaderNext是一个邮件附件下载器，基于Textual框架。

计划支持/已实现的功能：
- [x] 识别重复邮件
- [x] 并发运行
- [x] 搜索和下载过程可中断
- [x] 解析BODYSTRUCTURE以获取邮件结构，避免读取整封邮件
- [x] 较详细的错误信息
- [ ] 邮件过滤器
- [ ] 下载分类
- [ ] 普通附件下载
- [ ] 超大附件下载
  - [x] QQ邮箱(2种)
  - [x] 163邮箱(2种)
  - [x] 新浪邮箱
- [ ] 日志
- [ ] 多语言支持
